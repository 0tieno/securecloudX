# v1.2 — Weekly Email Digest

- Released: 12th May 2026
- Branch: `ft-send-automated-mails`

## What's new

- Automated weekly email digest sent every Tuesday at 3pm EAT (12:00 UTC) to all subscribed users
- Emails include personalised progress summary, next module CTA, featured blog post, and a statement of the week
- One-click unsubscribe via token-based `/unsubscribe` route
- New `/unsubscribe` page in the app
- AuthContext now seeds `email_preferences` for every new sign-in

## Infrastructure

- Two Supabase Edge Functions deployed: `send-weekly-email`, `unsubscribe`
- Three new DB tables: `email_preferences`, `weekly_content`, `email_log`
- pg_cron + pg_net extensions enabled; cron job registered as `weekly-email-digest`
- Email delivery via Resend from `progress@securecloudx.xyz`
- `List-Unsubscribe` and `List-Unsubscribe-Post` headers added for Gmail/Yahoo bulk sender compliance
